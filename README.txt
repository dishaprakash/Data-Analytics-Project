README
The code uses certain dependencies which needs to be installed before running it
 - sklearn
 - matplotlib
 - scipy
 - numpy
 - pandas
 - seaborn

The code is divided into components:
 - Data visualization
 - Modelling

Data visualization shows the prevelent trends in the dataset.
One of the main inference from this is the people of color column which consists of 
Hispanics, Asians, etc. and their incomes.

The model building includes building 3 models to compare the results 
and base a hypothesis on this.
